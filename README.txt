===========================
 LunaProxy C-Project (Mock)
===========================

Dies ist ein symbolisches Projekt für ein C-basiertes LunaProxy-API-Hosting.

Hinweis: Dieses Projekt simuliert nur ein C-Programm. Für echtes HTTP-Serving in C
müsstest du Libraries wie libmicrohttpd, mongoose oder CivetWeb einbinden.

Build:
    gcc main.c -o lunaproxy

Run:
    ./lunaproxy
